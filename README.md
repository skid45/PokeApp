# PokeApp

Приложение сделано для тестового задания от VK.

Использованный стек:
* Jetpack Compose
* Navigation Component
* Multimodule (api/impl)
* Clean arch. + MVVM(+)
* Dagger2
* OkHttp, Retrofit, Gson
* Coroutines, Flow
* Paging3
* Coil
